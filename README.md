# API Gateway

## Requirement
- [nodejs](https://nodejs.org/dist/v16.17.0/node-v16.17.0-x64.msi)
- [git](https://git-scm.com/download/win)

## Install

```
// 1. Clone Project
git clone https://github.com/ndiing/server.git

// 2. 
cd server

// 3. Install
npm install

// 4. Run
npm start

// 5. For the next update
npm update
```
