# Install

```
// 1. Clone project
git clone https://github.com/ndiing/server.git

// 2. Change directory
cd server

// 3. Install packages
npm install

// 4. Start server
npm start

// Done

// 1. Install any update
npm update
// 2. and then start again
npm start
```
