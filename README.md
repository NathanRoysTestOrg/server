# Install

1. Clone project
    ```
    git clone https://github.com/ndiing/server.git
    ```
1. Change directory
    ```
    cd server
    ```
1. Install packages
    ```
    npm install
    ```
1. Start server
    ```
    npm start
    ```
1. Install any update
    ```
    npm update
    
    // and then start again
    npm start
    ```
