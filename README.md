# Server

reverse proxy gateway

## Usage

install with built in server

```
// 1. clone project
git clone https://github.com/ndiing/server.git

// 2. change directory
cd server

// 3. install packages
npm install

// 4. run server
npm start

// 5. when any update
npm update

// and then start over
```
